# Icon Placeholder Files

This directory should contain the following icon files:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Generating Icons

### Option 1: Using the HTML Generator
1. Open `generate-icons.html` in your browser
2. Right-click each canvas
3. Save as PNG with the appropriate filename in this directory

### Option 2: Use an Icon Generator Tool
Use online tools like:
- https://www.favicon-generator.org/
- https://www.iconion.com/
- Or any graphic design software (Figma, Photoshop, etc.)

### Design Guidelines
- Use a simple, recognizable symbol (play button, video icon, etc.)
- Use the brand colors: Purple gradient (#667eea to #764ba2)
- Ensure the icon is visible at small sizes (16x16)
- Use transparency or solid background
