# ReWatch Roadmap

_Last updated: November 2, 2025_

## Short-term (Q4 2025)

- Ship detector telemetry dashboard and debug toggle in the popup.
- Publish beta program guidelines and enrollment workflow.
- Produce lightweight marketing landing bundle for docs site.
- Harden detector metadata extraction for Brocoflix edge cases.
- Add automated smoke scripts for content tracker heartbeat checks.

## Near-term (H1 2026)

- Group watch history by series with season progress indicators.
- Introduce optional local backups with scheduled exports.
- Expand detector coverage to Roku Channel and YouTube (beta).
- Add detector sandbox replay harness for DOM fixture regression testing.
- Implement multi-profile configuration sync via Chrome identities.

## Long-term (H2 2026 and beyond)

- Explore opt-in community rulesets for emerging platforms.
- Add advanced analytics: streaks, total watch time, discovery insights.
- Investigate secure cloud sync connectors (self-hosted storage targets).
- Integrate watchdog alerts for detector breakages with automatic fallbacks.
- Research cross-browser support starting with Manifest V3-compatible forks.
